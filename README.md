## 🤝 Como Contribuir

Para adicionar um novo link:

1. Escolha a categoria apropriada.
2. Use o formato: `- Nome do Recurso – Descrição curta.`
3. Faça um Pull Request com sua sugestão.

Exemplo:
- Figma – Ferramenta de design colaborativo.
